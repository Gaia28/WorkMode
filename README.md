# Workmode - Automation script
A pratical script to automation initialization of PHP enviroment on linux

## What it does?
This script save time when initializing whit one command this programs:
- VsCode
- Firefox
- Lampp
- Spotify
## How to install?
- clone:
``
git clone https://github.com/Gaia28/WorkMode.git
``
- Move the script to your binary directory: ``
 sudo cp devmode.sh /usr/local/bin/
``
- Give execution permition: ``
 chmod +x devmode.sh
``
## How to use?
Run ``devmode.sh`` on terminal
